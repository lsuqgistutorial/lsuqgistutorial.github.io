Importing GNSS Data to QGIS

Almost all GNSS receivers will save the data in a format called GPX (.gpx). QGIS can open this type of data directly. Therefore, all we need to do is access the receiver, select the files and open then in QGIS.
 
First step is to connect the receiver to your computer. Consider looking at the receiver manual abou the process of getting to the GPX folder and exporting these files. If you want to practice along this tutorial HERE you can download the files used.
 
After exporting the GPX files from your receiver, you just need to start QGIS and by "dragging and dropping" the file into QGIS a window such as the one below should pop up. From available data, you can access the folder "Tracks" and use one of the two ".gpx" files.
 
 
 
 
As you can see, you have two types of geometries (look at the "Geometry type" column), points and line. Also, we have 98 track points and 1 track. Let's add all the layers - even though some will be empty -  and see what happens.
