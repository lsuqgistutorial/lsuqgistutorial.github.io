GNSS Data Imported to QGIS

 

Imported Data

Once imported, points and lines should appear at your canvas
Top

Layer Panel

In your layer panel the desciption of each of the layers added should be displayed. The colors of each vector matches with the colors displayed in the canvas
Top
 

EPSG:4326 button

THE MOST IMPORTANT! The canvas CRS (Coordinate Reference System) should match with the receiver CRS Most common to be used is WGS84 decimal degress (EPSG:4326). Hoiwever, it is important to check! If you want ot know more about CRS, check this LINK
Top
 
Now, by checking and unchecking the "box" in front  of each layer description, you will be able to hide or not the layer from your canvas. If you imported all the layer without choosing the only one that had data, some of your layers will be empty.
