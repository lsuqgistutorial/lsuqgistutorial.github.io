Exporting Layers

Exporting the information is a very important process. First because if you need to share, open this data in other softwares, you might need to save it in a different format. second, because if we want to edit (e.g. remove points, change the track) the layer, we cannot do it while the format is still GPX. We need to convert it, process that can be done by exporting the layer in a different format.
 
To export a specific layer:
 
  • Right click over the layer description in the Layers Panel -  The Menu presented below should appear
     
 

Export Option

To export you will need to scroll down to the Export option and choose "Save Feature as". Then a window such as the one below will be presented
Top
 
  • In such window the most important spots are: "Format", "File name" and "CRS"
  • In Format you will be able to choose among different types of formats. If you want to edit the layer, you should save as "ESRI Shapefile" or "GeoPackage". However, other formats are available, including CSV (allow the data to be opened using Excel - this feature will be explored later)
     
  • File name: do not get trapped and try just to add the name you want for the file in the blank space. You need to set a directory for the file. Therefore, click in  and choose where you want to save and the name of the file.
     
  • CRS:  as mentioned before, the coordinate reference system is very important, at this point, if you want to convert your CRS from WGS84 Decimal Degrees to WGS84 UTM, you can! You just need to select the right CRS.
 
 
After you are done with your file format, name and CRS, you just need to click and your file will be saved. Also, note ththat the check box at the LEFT of the "Ok" button says  . If that box is checked, your new file will be automatically be added to the map.
